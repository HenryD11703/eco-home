# Notas desarrollo

Notas de cosas que aprendi en el desarrollo de EcoMarket 

mongoose permite definir esquemas para los documentos en la base de datos, lo que facilita la validación y la estructura de los datos.

en el esquema mismo se pueden definir validaciones, valores por defecto y otros comportamientos que se aplican a los documentos de esa colección.

y entonces no es un objeto plano, sino que tiene un comportamiento más rico gracias a los métodos que se pueden definir en el esquema.

y este lo mete en la base de datos? 
respuesta: mongoose permite interactuar con la base de datos de manera sencilla, proporcionando métodos para crear, leer, actualizar y eliminar documentos.

en este proyecto voy a hacer un ecommerce de una tienda de muebles ecologicos

entonces en la base de datos solo voy a guardar lo que son usuarios y productos ya que simulare un checkout sin necesidad de guardar la información de pago.
